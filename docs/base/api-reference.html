API Reference

This page provides a high-level overview of the main APIs available in FlexNet JSX. For detailed information, please see the specific API pages.

Core Functions

Utilities for functional programming.

  • pipe(...fns): Left-to-right function composition.
  • compose(...fns): Right-to-left function composition.
  • curry(fn): Curries a function.
Go to Core Functions API →

Core Types

Data structures for safe and explicit data handling.

  • Maybe: For optional values (Just or Nothing).
  • Either: For values that can be one of two types (Left or Right).
  • Result: For operations that can succeed or fail (Ok or Error).
Go to Types API →

Render System

Functions for creating and rendering UI elements.

  • createElement(type, props, ...children): Creates a virtual DOM node.
  • render(vNode, container): Renders a virtual DOM tree to a real DOM container.
  • Fragment: A component for grouping elements without a wrapper node.
Go to Render API →

State System

Functions for centralized state management.

  • createStore(reducer, initialState, effects): Creates the application store.
  • store.getState(): Gets the current state.
  • store.dispatch(action): Sends an action to the reducer.
  • store.subscribe(listener): Listens for state changes.
Go to Store API →
