Getting Started Guide

This guide walks you through the initial setup, core concepts, and best practices for starting with the FlexNet Framework.

  • Project Setup: Directory structure guidance and initial project configuration steps.
  • Core Concepts: Pure functions, type safety with Maybe/Either/Result, immutable state, and effect handling.
  • Development Workflow: Process for feature development, testing strategies, and documenting code.
  • Best Practices: Recommendations on file organization, code style, testing methodologies, and performance.
