Utility Functions

FlexNet includes a small set of powerful utility functions for composition and data manipulation. These are the tools you'll use to build complex logic from simple, reusable pieces.

pipe(...fns)

Performs left-to-right function composition. It takes a sequence of functions and returns a new function. When the new function is called, it passes its arguments to the first function, the result of that to the second, and so on. 

import { pipe } from './src/core/functions.js';

const add5 = x => x + 5;
const multiplyBy2 = x => x * 2;

const calculate = pipe(add5, multiplyBy2);

// is equivalent to: multiplyBy2(add5(10))
const result = calculate(10); // 30

compose(...fns)

Performs right-to-left function composition. It's the same as pipe, but the functions are applied in the reverse order. 

import { compose } from './src/core/functions.js';

const add5 = x => x + 5;
const multiplyBy2 = x => x * 2;

const calculate = compose(multiplyBy2, add5);

// is equivalent to: multiplyBy2(add5(10))
const result = calculate(10); // 30

The choice between pipe and compose is a matter of preference. pipe often feels more natural as it follows the order of reading.

curry(fn)

Transforms a function that takes multiple arguments into a sequence of functions that each take a single argument. This is useful for creating specialized functions from more generic ones. 

import { curry } from './src/core/functions.js';

const add = (a, b) => a + b;
const curriedAdd = curry(add);

const add10 = curriedAdd(10); // Creates a new function that adds 10

const result = add10(5); // 15
