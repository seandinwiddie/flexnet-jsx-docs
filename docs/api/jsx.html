API: JSX & Rendering

This section details the core functions for creating UI elements and rendering them to the DOM.

createElement(type, props, ...children)

Creates and returns a new FlexNet element (a virtual DOM node).

  • type: A string (e.g., 'div', 'p') or a FlexNet component function.
  • props: An object containing properties for the element.
  • children: A list of child elements or text nodes.
createElement('h1', { className: 'title' }, 'Hello');

render(element, container)

Renders a FlexNet element into a DOM container, and handles updating the DOM on subsequent calls.

  • element: The FlexNet element to render.
  • container: The DOM node to render the element into.
render(
    createElement(App),
    document.getElementById('root')
);

Fragment

A component that lets you group a list of children without adding extra nodes to the DOM.

<>
    <ChildA />
    <ChildB />
</>
