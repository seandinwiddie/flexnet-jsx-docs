API: Result

The Result type is for operations that can succeed or fail. It is either an Ok or an Error.

Result.Ok(value)

Creates a Result representing a successful operation.

Result.Error(value)

Creates a Result representing a failed operation.

Result.fromTry(fn)

Executes a function that might throw an exception and wraps it in a Result.

result.isOk()

Returns true if the Result is Ok.

result.isError()

Returns true if the Result is an Error.

result.map(fn)

Applies a function to the value inside an Ok result.

result.unwrapOr(defaultValue)

Returns the value from an Ok or the provided default if it's an Error.
