Maybe Type

The Maybe type represents optional values in a functional way. It is used to handle potentially missing values (null or undefined) without relying on null checks or conditionals.

Maybe.Just(value)

Function → Maybe

Creates a Maybe type that contains a value.

Arguments

Name Type Description
value Any The value to wrap in a Maybe.Just

Returns

A Maybe.Just object with the following methods:

  • type - The string "Just"
  • value - The wrapped value
  • map(fn) - Apply a function to the wrapped value and return a new Maybe.Just
  • chain(fn) - Apply a function that returns a Maybe to the wrapped value
  • getOrElse(defaultValue) - Return the wrapped value

Example

import Maybe from '../core/types/maybe.js';

const justFive = Maybe.Just(5);
console.log(justFive.type);  // "Just"
console.log(justFive.value); // 5

Maybe.Nothing()

Function → Maybe

Creates a Maybe that represents no value.

Returns

A Maybe.Nothing object with the following methods:

  • type - The string "Nothing"
  • map(fn) - Returns Maybe.Nothing regardless of the function
  • chain(fn) - Returns Maybe.Nothing regardless of the function
  • getOrElse(defaultValue) - Returns the defaultValue

Example

import Maybe from '../core/types/maybe.js';

const nothing = Maybe.Nothing();
console.log(nothing.type);              // "Nothing"
console.log(nothing.getOrElse("Default")); // "Default"

Maybe.fromNullable(value)

Function → Maybe

Creates a Maybe from a value that might be null or undefined.

Arguments

Name Type Description
value Any The value to check and wrap in a Maybe

Returns

A Maybe.Just if the value is not null or undefined, otherwise a Maybe.Nothing.

Example

import Maybe from '../core/types/maybe.js';

const validValue = Maybe.fromNullable("Hello");
console.log(validValue.type);  // "Just"

const nullValue = Maybe.fromNullable(null);
console.log(nullValue.type);   // "Nothing"

const undefinedValue = Maybe.fromNullable(undefined);
console.log(undefinedValue.type);  // "Nothing"

maybe.map(fn)

Method → Maybe

Apply a function to the value inside a Maybe (if it exists) and return a new Maybe.

Arguments

Name Type Description
fn Function The function to apply to the wrapped value

Returns

A new Maybe with the transformed value if the original Maybe was a Just, otherwise returns Nothing.

Example

import Maybe from '../core/types/maybe.js';

const justFive = Maybe.Just(5);
const justTen = justFive.map(x => x * 2);
console.log(justTen.value); // 10

const nothing = Maybe.Nothing();
const stillNothing = nothing.map(x => x * 2);
console.log(stillNothing.type); // "Nothing"

maybe.chain(fn)

Method → Maybe

Apply a function that returns a Maybe to the value inside a Maybe (if it exists).

This is also known as flatMap or bind in other functional libraries.

Arguments

Name Type Description
fn Function A function that takes a value and returns a Maybe

Returns

The Maybe returned by the function if the original Maybe was a Just, otherwise returns Nothing.

Example

import Maybe from '../core/types/maybe.js';

// Safe division function that returns Maybe
const safeDivide = (a, b) => 
  b === 0 ? Maybe.Nothing() : Maybe.Just(a / b);

const justTen = Maybe.Just(10);
const result = justTen.chain(x => safeDivide(x, 2));
console.log(result.value); // 5

const zeroDivision = justTen.chain(x => safeDivide(x, 0));
console.log(zeroDivision.type); // "Nothing"

// Chain operations can be... chained
const user = { 
  profile: { 
    name: "Alice" 
  } 
};

const getName = obj => 
  Maybe.fromNullable(obj)
    .chain(o => Maybe.fromNullable(o.profile))
    .chain(p => Maybe.fromNullable(p.name))
    .getOrElse("Anonymous");

console.log(getName(user)); // "Alice"
console.log(getName(null)); // "Anonymous"
console.log(getName({profile: null})); // "Anonymous"

maybe.getOrElse(defaultValue)

Method → Any

Extract the value from a Maybe, providing a default value for the Nothing case.

Arguments

Name Type Description
defaultValue Any The value to return if the Maybe is Nothing

Returns

The wrapped value if the Maybe is a Just, otherwise the defaultValue.

Example

import Maybe from '../core/types/maybe.js';

const justHello = Maybe.Just("Hello");
console.log(justHello.getOrElse("Default")); // "Hello"

const nothing = Maybe.Nothing();
console.log(nothing.getOrElse("Default")); // "Default"

// Real-world example: safe property access
const getProperty = (obj, prop) => 
  Maybe.fromNullable(obj)
    .chain(o => Maybe.fromNullable(o[prop]))
    .getOrElse("Property not found");

const user = { name: "Alice", age: 30 };
console.log(getProperty(user, "name")); // "Alice"
console.log(getProperty(user, "email")); // "Property not found"
console.log(getProperty(null, "name")); // "Property not found"

Maybe.map(fn)

Static Function → Function

A curried version of map that can be used with function composition.

Arguments

Name Type Description
fn Function The function to apply to the wrapped value

Returns

A function that takes a Maybe and returns a new Maybe with the transformed value.

Example

import Maybe from '../core/types/maybe.js';
import { compose } from '../core/functions/composition.js';

const double = x => x * 2;
const addTen = x => x + 10;

const maybeDouble = Maybe.map(double);
const maybeAddTen = Maybe.map(addTen);

const justFive = Maybe.Just(5);
const result = maybeAddTen(maybeDouble(justFive));
console.log(result.value); // 20

// Using compose for more readable chaining
const transformValue = compose(
  Maybe.map(addTen),
  Maybe.map(double)
);

const result2 = transformValue(Maybe.Just(5));
console.log(result2.value); // 20

Maybe.chain(fn)

Static Function → Function

A curried version of chain that can be used with function composition.

Arguments

Name Type Description
fn Function A function that takes a value and returns a Maybe

Returns

A function that takes a Maybe and returns the result of applying the function to the wrapped value.

Example

import Maybe from '../core/types/maybe.js';
import { compose } from '../core/functions/composition.js';

// A function that returns a Maybe
const safeSqrt = x => 
  x >= 0 ? Maybe.Just(Math.sqrt(x)) : Maybe.Nothing();

const maybeChainSqrt = Maybe.chain(safeSqrt);

const justTwentyFive = Maybe.Just(25);
const result = maybeChainSqrt(justTwentyFive);
console.log(result.value); // 5

// Chain with function composition
const safeDouble = x => Maybe.Just(x * 2);

const processValue = compose(
  Maybe.chain(safeSqrt),
  Maybe.chain(safeDouble)
);

const result2 = processValue(Maybe.Just(16));
console.log(result2.value); // 4

Maybe.getOrElse(defaultValue)

Static Function → Function

A curried version of getOrElse that can be used with function composition.

Arguments

Name Type Description
defaultValue Any The value to return if the Maybe is Nothing

Returns

A function that takes a Maybe and returns either the wrapped value or the default value.

Example

import Maybe from '../core/types/maybe.js';
import { compose } from '../core/functions/composition.js';

const getOrZero = Maybe.getOrElse(0);

const justFive = Maybe.Just(5);
console.log(getOrZero(justFive)); // 5

const nothing = Maybe.Nothing();
console.log(getOrZero(nothing)); // 0

// Real-world example with compose
const safeParse = str => {
  try {
    return Maybe.Just(JSON.parse(str));
  } catch (e) {
    return Maybe.Nothing();
  }
};

const getConfig = compose(
  Maybe.getOrElse({ default: true }),
  Maybe.chain(obj => Maybe.fromNullable(obj.config)),
  safeParse
);

const validJSON = '{"config": {"debug": true}}';
console.log(getConfig(validJSON)); // {debug: true}

const invalidJSON = '{not valid json}';
console.log(getConfig(invalidJSON)); // {default: true}

Common Use Cases

Safe Property Access

import Maybe from '../core/types/maybe.js';

// Function to safely access nested properties
const getNestedProp = (obj, ...props) => {
  return props.reduce(
    (acc, prop) => acc.chain(o => Maybe.fromNullable(o[prop])),
    Maybe.fromNullable(obj)
  );
};

const user = {
  profile: {
    address: {
      city: "San Francisco"
    }
  }
};

// Safe property access without "&&" chains or try/catch
const city = getNestedProp(user, "profile", "address", "city")
  .getOrElse("Unknown City");

console.log(city); // "San Francisco"

const zipCode = getNestedProp(user, "profile", "address", "zipCode")
  .getOrElse("No Zip Code");

console.log(zipCode); // "No Zip Code"

Form Validation

import Maybe from '../core/types/maybe.js';

// Form validation functions
const validateEmail = email => {
  const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return regex.test(email) ? Maybe.Just(email) : Maybe.Nothing();
};

const validatePassword = password => {
  return password.length >= 8 
    ? Maybe.Just(password) 
    : Maybe.Nothing();
};

// Validate form data
const validateForm = formData => {
  const emailResult = Maybe.fromNullable(formData.email)
    .chain(validateEmail);
    
  const passwordResult = Maybe.fromNullable(formData.password)
    .chain(validatePassword);
    
  // Both must be valid
  return emailResult.chain(email => 
    passwordResult.map(password => ({ email, password }))
  );
};

// Valid form
const validForm = { 
  email: "user@example.com", 
  password: "securepass123" 
};

const validResult = validateForm(validForm);
console.log(validResult.type); // "Just"

// Invalid form
const invalidForm = { 
  email: "not-an-email", 
  password: "short" 
};

const invalidResult = validateForm(invalidForm);
console.log(invalidResult.type); // "Nothing"
