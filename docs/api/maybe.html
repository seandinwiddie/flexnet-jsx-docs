API: Maybe

The Maybe type is used to represent an optional value. See the guide on the type system for more details.

Maybe.Just(value)

Creates a Maybe containing a value.

Maybe.Nothing()

Creates a Maybe representing the absence of a value.

maybe.isJust()

Returns true if the Maybe is a Just.

maybe.isNothing()

Returns true if the Maybe is a Nothing.

maybe.map(fn)

Applies a function to the value inside a Just and returns a new Just. If it's Nothing, it returns Nothing.

maybe.unwrapOr(defaultValue)

Returns the value inside a Just or returns the defaultValue if it's Nothing.
