API: State Management (Store)

The store is the object that holds your application's state. See the guide on state management for more details.

createStore(reducer, initialState, effects = [])

Creates the central state store for the application.

  • reducer: The main reducer function.
  • initialState: The starting state of the application.
  • effects: An optional array of effect functions (middleware).

store.getState()

Returns the current state tree of your application.

store.dispatch(action)

Dispatches an action. This is the only way to trigger a state change.

store.subscribe(listener)

Adds a change listener that will be called whenever an action is dispatched. Returns an unsubscribe function.
