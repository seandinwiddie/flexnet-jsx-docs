Installing FlexNet JSX

FlexNet JSX is designed for a straightforward setup with zero external dependencies. The recommended way to start is by using one of our pre-configured starter projects.

Core Principle: Zero Dependencies

FlexNet JSX uses only browser-native implementations and has no reliance on package managers like NPM or Yarn. Installation is as simple as getting the code and serving it.

Using the Starter Projects (Recommended)

The starter projects provide a complete, correctly structured foundation for your application. They are the quickest and most reliable way to begin developing with FlexNet JSX.

Step 1: Get the Code

First, clone the official FlexNet JSX repository to get access to the starter projects and the full framework source.

git clone https://github.com/flexnet/flexnet-jsx.git
cd flexnet-jsx
